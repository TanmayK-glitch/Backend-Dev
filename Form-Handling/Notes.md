**Sessions and Cookies**
*Sesssions*
- It is a way to remember who the user is after login (without saving passord anywhere).
- It creates a random ID for that particular User and saves all the info of the user wiht that ID. 
- Login to Logout (1 Session).

*Cookies*
- Small piece of information a server asks the browser to store.
- After storing every request you make will includes that stored info. (Theme, cart items, login tokens).

*----*
- The data can be sent in many formats to the Database / Server: 
1. JSON (if using API)
2. Key-Value Pairs (Sting)
3. Arrays (in PHP)
